微信小程序

微信小程序开发文档

原生框架 mina框架

小程序与web的区别

结构 web 小程序
页面 HTML WSML
样式 CSS WXSS
逻辑 Javascript Javascript
配置 无 Json

配置文件

全局配置 app.json

页面pages

显示小程序中拥有的页面路径

当我们想要自己去创建一个页面的时候，我们可以直接在配置文件中添加路径并保存，就可以自动的生成页面及其相关配置

设置首页面

pages中的第一个页面路径就是首页(谁在第一个谁就是首页面)

开发时,尽量将当前开发的页面设置成首页,因为每次保存后默认都会跳到首页面

window

https://developers.weixin.qq.com/miniprogram/dev/reference/configuration/app.html#window

导航栏相关配置

navigationBarBackgroundColor

navigationBarTitleText(标题名称)

navigationBarTextStyle(标题字体颜色)

字体颜色只能是black或white

backgroundTextStyle(下拉刷新时的加载样式)

默认是light

需要开启下拉设置后才能生效”enablePullDownRefresh”:true

改为dark

tabBar(导航栏)

list

list用来设置导航栏每个选项的样式

list中至少要包含两个才能显示

color(字体颜色)

与list同级目录下使用color

页面配置

可以在pages下的各个页面的json文件中自定义属性

标签

text相当于html的span，不会自动换行

view相当于html的div，会自动换行

checkbox 复选框,和以前一样

text

文本

view

块

checkbox

复选框

checked默认为true

数据绑定

view属性

data-num

传布尔类型时不要写成string类型(粗心),而且使用布尔类型时双引号里不要有空格,否则也按字符串处理

运算

判断

for循环遍历(列表渲染)

首先在js中创建一个对象数组

然后使用wx:for=”“来遍历，使用wx:for-item=”遍历名称”来设置遍历的名称，用wx:for-index=”索引名”来定义索引名(索引默认从0开始)

设置唯一属性，相当于sql中的唯一主键

一层循环

如果是一层循环可以不写wx:for-item和wx:for-index

循环名称用item，索引用index

遍历对象

index是key，item是value

判断(条件渲染)

wx:if=“”

可以用来显示与隐藏内容

双向绑定

this.data.

获取当前data中的数据

this.detail.value

获取input中的值

input

文档

bindinput=“函数名” 输入数据是触发

input中的type对应着输入信息时键盘的模式

var let const

var是全局变量

let是局部变量

const是只读常量

wxcc

rpx与css的px不同点在于,px设置的大小不会随着页面的大小改变而改变,而rpx可以随着页面的大小而改变

导入文件

通过@import来引入,只支持相对路径
